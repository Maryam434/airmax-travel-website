# ✈️ Travel Booking Website 

A full-stack Travel Booking Website developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL. The platform allows users to explore travel destinations, create accounts, log in securely, and make travel bookings that are stored in a MySQL database.

## Features

- User Registration and Login System
- Travel Destination Listings
- Travel Booking Functionality
- Secure Data Storage with MySQL
- Responsive User Interface
- Contact Section
- Fast and Interactive Experience

## Technologies Used

### Frontend

- HTML5
- CSS
- JavaScript

### Backend

- Node.js
- Express.js

### Database

- MySQL

## Installation

```bash

npm install

```

## Database Setup

1. Create a MySQL database.
2. Import the required tables.
3. Update database credentials in the project configuration.

## Run Project

```bash

node server.js

```
